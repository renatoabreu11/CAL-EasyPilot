INSTRUÇÕES DE COMPILAÇÃO e DEPENDÊNCIAS

1. Criar um projeto vazio C++, com MinGW como Toolchain.
2. Adicionar todos os ficheiros da pasta EasyPilot ao projeto.
3. Para correr o GraphViewer, alterar nas propriedades do projeto: C++ Build > Settings > MinGW C++ Linker > Miscellaneous > Adicionar outro objeto > <MinGW Path>\libwsock32.a (ex: C:\MinGW\lib\libwsock32.a).
4. Adicionar "-static" às flags do linker.


É necessário ter o JAVA instalado para correr o GraphViewerController.jar.
